# SkyWings Frontend

React + Vite frontend for the SkyWings Aviation & Travel booking platform.

## Quick Start

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Technologies

- React 18.3
- Vite 6
- Tailwind CSS
- React Router DOM
- Lucide React Icons

## Features

- Modern, responsive UI
- Fast page loads with Vite
- Smooth animations
- Mobile-first design
- Component-based architecture

## Build

```bash
npm run build
```

Output will be in the `dist/` folder, ready for deployment.
